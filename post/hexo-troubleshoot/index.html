复现

每次 hexo s 部署后，访问页面均会有

解决方案

Hexo 这里的 warning是由于 stylus 导致的，幸运的是stylus 在 0.54.8 版本修复了这个问题，所以对于 Hexo 用户来说，重新装一下hexo-renderer-stylus，就可正常使用。 但是我重装之后发现还是会报警告，继续追溯源头，发现是这其实是 ni b这个包里的 stylus 引起的问题，而这个包已经很久没更新了。 2.1 将hexo-renderer-stylus更新到2.0.1 2.2 将stylus 从0.54.5更新到0.54.8

添加内容至根目录 package.json ：

1
2
3

"resolutions": {
  "stylus": "^0.54.8"
}
